# 📊 Análise de Performance: Rede de Lojas do Sr. João

Este projeto consiste em uma análise exploratória de dados de vendas de quatro unidades comerciais, com o objetivo de fornecer um embasamento estratégico para a tomada de decisão sobre o desinvestimento de uma das unidades.

## 🚀 Tecnologias Utilizadas
* **Python 3**
* **Pandas**: Manipulação e agregação de dados.
* **Seaborn & Matplotlib**: Visualização de dados e gráficos de performance.
* **IPython.display**: Relatórios formatados em HTML/Markdown.

## 📈 Conclusão da Análise
A recomendação final foi a **venda da Loja 4**. 
Apesar de apresentar a maior eficiência logística (menor custo de frete), a unidade registrou o menor faturamento da rede, indicando que sua operação não escala o suficiente para justificar a manutenção em comparação às outras filiais mais rentáveis.
